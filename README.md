# Hospital Management System

A comprehensive web-based Hospital Management System developed using Django that streamlines hospital operations, patient management, doctor scheduling, and appointment booking through an intuitive and user-friendly interface.

## 📌 Project Overview

The Hospital Management System is designed to automate and simplify the daily operations of healthcare institutions. It provides a centralized platform for managing patients, doctors, appointments, and medical records efficiently.

This project demonstrates the implementation of full-stack web development concepts using Django and follows a structured approach to data management and user authentication.

## ✨ Features

### Patient Management

* Patient registration and profile management
* View patient records and information
* Secure storage of patient details

### Doctor Management

* Add and manage doctor profiles
* Track doctor specialization and availability
* Manage doctor schedules

### Authentication & Security

* Secure user registration and login
* Role-based access control
* Session management and authentication

### Administrative Dashboard

* Manage doctors, patients, and appointments
* Monitor hospital activities
* Centralized data management

## 🚀 Installation and Setup

### Clone the Repository

```bash
git clone https://github.com/yourusername/hospital-management-system.git
```

### Navigate to Project Directory

```bash
cd hospital-management-system
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Virtual Environment

Windows:

```bash
venv\Scripts\activate
```

Mac/Linux:

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Apply Migrations

```bash
python manage.py makemigrations
python manage.py migrate
```

### Run the Development Server

```bash
python manage.py runserver
```

### Open in Browser

```text
http://127.0.0.1:8000/
```
